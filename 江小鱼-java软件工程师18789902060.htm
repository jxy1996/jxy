项目名称：去哪儿旅游管理系统            

 

开发平台：Windows 7

 

开发工具：Eclipse , MySQL 

 

设计相关技术：SSM框架、JavaWeb、JS、Tomcat

 

项目描述：

1.该项目采用了MVC的设计模式：

业务层（M）：Service 组件

数据层（M）：Dao组件

表现层（V）：html+css+jQuery

控制层（C）：Controller 组件

 

2.项目分为两大模块：

产品管理模块和系统管理模块，其中产品管理模块又分为：项目管理，团目管理，产品分类，产品管理。

 

责任描述：

1.     项目管理：对产品增改查询以及禁用启用以及利用MyBatis进行动态SQL 进行模糊查询。

2.     团目管理：基于旅游产品的前提下对产品进行组团。

3.     产品分类：采用自关联表的方式进行查询，清楚上下级的关系。

4.     附件上传/下载：采用MD5加密的方式对上传文件描述进行加密，并将信息插入到数据库，以及文件的下载。
